Detailed Description

Author
joH1

This file contains the definition of a simple logging system.

Each call to a logging function will output its message – given as a format string and parameters.

A message is provided with a priority level, and can be filtered out, by specifiying a filter level.

Enumeration Type Documentation

◆ LogLevel

enum LogLevel

Defines the level of priority of a logging message, or an alias to set the log filter.

The message is to be filtered out from logs if its priority level is inferior to the one set in log_setfilter(). It will also be preceded in logs by a header, consisting of the name of the level and two separating dashes, e.g.: 

INFO -- info message content
Enumerator
LOG_DEBUG 

A message used while developing, to help the programmer

LOG_VERBOSE 

A detailled information message

LOG_INFO 

An information message

LOG_WARNING 

Denotes a malfunction in the system

LOG_ERROR 

Denotes a severe unexpected behavior which can lead to system failure

LOGF_ALL 

The maximal allowance level in filter

LOGF_NONE 

The most quiet level (only error messages are output)
